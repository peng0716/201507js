#网站的访问流程
1. 在浏览器中输入URL地址。
2. 把域名转成IP地址。
3. 客户端与服务器建立连接。
4. 客户端把请求发给服务器。
5. 服务器接收到请求把响应发给客户端。
6. 客户端接收响应，进行显示。
